Projetos

RickPedia

Este projeto foi desenvolvido como parte de um teste técnico para a empresa Dorotech, utilizando a API pública do Rick and Morty. A aplicação permite ao usuário filtrar personagens por diferentes critérios, além de utilizar uma barra de busca para localizar personagens específicos pelo nome.

Ao clicar no nome de um personagem, um modal é exibido com informações detalhadas como espécie, origem, status e imagem.

React - TypeScript - Tailwind CSS

Clone AirBnb

Projeto desenvolvido durante um curso, com o objetivo de praticar tecnologias modernas de front-end. Trata-se de um clone da página inicial do Airbnb, construído com React + TypeScript, usando Next.js para rotas e renderização server-side, e Tailwind CSS para a estilização responsiva.

Os dados das propriedades são consumidos de uma API fornecida pela Codans, simulando listagens reais.
